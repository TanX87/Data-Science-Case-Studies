# Credit Risk Modeling: Probability of Default Prediction

## Overview

This project develops a credit risk model to predict the probability of default of borrowers using statistical modeling and machine learning techniques.

The objective is to identify the main factors associated with credit delinquency and build predictive models that can support risk assessment and decision-making processes in financial institutions.

## Methodology

The analysis follows a credit scoring approach, including:

- Data preprocessing and missing value treatment.
- Exploratory data analysis.
- Variable transformation using Weight of Evidence (WoE).
- Variable selection through Information Value (IV).
- Development of predictive models:
  - Logistic Regression.
  - Decision Tree.
  - Random Forest.
- Model evaluation using confusion matrices and ROC-AUC analysis.

## Key Results

The models were evaluated based on their ability to classify customers according to their default risk.

The logistic regression model achieved an accuracy of approximately 83% and a ROC-AUC of 82.4%, showing a good capacity to discriminate between customers with different credit risk profiles.

The analysis identified variables related to previous payment behavior as important predictors of default probability.

This project demonstrates the application of statistical and machine learning techniques in credit risk analysis, connecting predictive modeling with concepts used in Probability of Default (PD) estimation and risk management.
